# Milk WEB

[Milk-Fonum](https://github.com/KwonYH-sky/Milk-Fonum)을 재설계하며 만들어보는 데 목적을 둔다.

#### 목표
- 자유롭게 이야기 나누는 공간
- 일반적인 커뮤니티 사이트가 가지고 있는 서비스 구현
- 서비스 배포


#### 기술 스텍
- Back End
    - Spring Boot 3.1.2 (Java 17)
    - JPA
    - QueryDsl
- Front End
    - Thymeleaf
    - JQuery + Javascript(ES6)
    - Axios
- DB
    - MySQL
